# T-004 · 实现 Banner 轮播

**所属需求：** R-004
**优先级：** P0
**状态：** ✅ 已完成

## 任务描述

使用 ArkUI Swiper 组件实现 Banner 轮播，支持自动轮播和手动滑动。

## 验收标准

- [ ] Banner 高度 170px，圆角 16px
- [ ] 3 张 Banner 图片可轮播
- [ ] 自动轮播间隔 5 秒
- [ ] 底部有指示器（3 个小圆点，当前页高亮）
- [ ] Banner 上有标题、副标题、CTA 按钮（"立即抢购"）

## 技术方案

- 使用 `Swiper` 组件
- 手动实现指示器（Row + Circle）
- Banner 内容用绝对定位覆盖在图片上

## 依赖

- T-001（页面框架）

## 预估时间

1.5 小时
