# T-006 · 实现精选好物 Grid

**状态：** ✅ 已完成

## 任务描述

实现"精选好物"区域，2 列 Grid 布局展示商品卡片。

## 验收标准

- [ ] 标题"精选好物"SemiBold 18px
- [ ] 2 列 Grid，每个卡片 160×200px，圆角 14px
- [ ] 卡片内容：商品图片（136×110，圆角 10）+ 名称 + 价格
- [ ] 商品名称 14px SemiBold，价格 16px Bold Indigo 色
- [ ] 使用 ProductCard.ets 可复用组件
- [ ] 使用 Mock 数据

## 技术方案

- 使用 `Grid` + `GridItem` 组件
- 或 Flex 布局（Row + Column）
- ProductCard.ets 接收 Product 对象作为参数

## 依赖

- MockData.ets（商品数据）
