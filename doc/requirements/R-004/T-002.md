# T-002 · 实现底部导航组件

**所属需求：** R-004
**优先级：** P0

## 任务描述

实现 `BottomNavigation.ets` 可复用组件，4 个 Tab（首页/搜索/购物车/我的），支持选中状态切换。

## 验收标准

- [ ] 组件接收 `currentIndex` 和 `onTabChange` 参数
- [ ] 4 个 Tab 正确显示（图标 + 文字）
- [ ] 选中 Tab 文字为 Indigo 色，未选中为灰色
- [ ] 点击 Tab 触发 `onTabChange` 回调
- [ ] 组件高度 64px，背景白色，顶部有分割线
