# T-005 · 实现商品分类

**所属需求：** R-004
**优先级：** P0
**状态：** ✅ 已完成

## 任务描述

实现 4 个商品分类入口（服饰/数码/家居/运动），每个分类有彩色背景 + SVG 图标 + 标签。

## 验收标准

- [ ] 4 个分类横向排列，每个 72×72px
- [ ] 各有不同底色（Indigo-tint / Orange-tint / Green-tint / Violet-tint）
- [ ] 每个分类有 Lucide SVG 图标（28px）
- [ ] 图标下方有分类名称（12px，Medium）
- [ ] 分类间距均匀

## 技术方案

- 使用 Row + ForEach 布局
- SVG 图标用内嵌字符串方式

## 依赖

- T-001（页面框架）

## 预估时间

1 小时
