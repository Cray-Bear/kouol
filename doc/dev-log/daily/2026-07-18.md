### 14:02 - 扫描敏感文件 ❌ 翻车

- **操作：** 用 `Get-ChildItem` + `Where-Object` 查找敏感文件
- **目的：** 确保不会把 `.env`、`.p12`、`credentials` 等推送到 GitHub
- **结果：** 失败
- **遇到的问题：**
  - **现象：** `Get-ChildItem: command not found`
  - **排查：** 意识到 Claude Code 的 Bash 工具在 Windows 上内部是 bash（MSYS/MinGW），不是 PowerShell
  - **根因：** CLAUDE.md 说"Windows 环境，使用 PowerShell 语法"指的是 PowerShell 工具，Bash 工具内部仍是 bash
  - **解决：** 改用 `find` 命令
  - **给未来 AI 的提示：** ⚠️ **Claude Code 在 Windows 上有两个 shell**：`Bash` 工具内部是 bash（用 find/grep/awk），`PowerShell` 工具才是 powershell.exe（用 Get-ChildItem/Where-Object）。CLAUDE.md 说的"PowerShell 语法"适用于 PowerShell 工具，不适用于 Bash 工具。**执行命令前先确认用哪个工具**。
