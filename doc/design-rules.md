## 色彩系统

```
Primary (主色):    #4F46E5  indigo   → 按钮、价格、选中态、导航高亮
Accent (强调色):   #F97316  orange   → 促销标签、CTA 按钮、VIP 徽章
BG (背景):         #F9FAFB  gray-50  → 页面背景
Surface (卡片):    #FFFFFF  white    → 卡片、输入框背景
Text-1 (主文字):   #111827  gray-900 → 标题、重要文字
Text-2 (次要文字): #6B7280  gray-500 → 描述、辅助文字
Text-3 (弱文字):   #9CA3AF  gray-400 → 占位符、禁用态
Border (边框):     #E5E7EB  gray-200 → 分割线、卡片边框
Success:           #10B981  emerald  → 成功状态
Danger:            #EF4444  red      → 删除、错误
```

## 字体规范

- **字体家族**: Inter
- **标题**: Bold, 22-27px (页面标题、价格)
- **副标题**: Semi Bold, 17-19px (栏目名、按钮文字)
- **正文**: Regular, 15-17px (描述、列表项)
- **辅助**: Regular, 13-15px (标签、辅助说明)
- **行高**: 1.5 倍

## 组件规范

### 状态栏 (Status Bar)
- y=0, h=44
- 左侧时间 "9:41" at x=24, y=12
- 右侧电池图标 at x=329, y=17
- 白色背景

### 导航栏 (Nav Bar)
- y=44-100, h=56 (居中内容 at y=62)
- 返回箭头: 22x22 SVG, x=20
- 标题: Bold 19px, 居中 x≈143
- 右侧操作: 20x20 SVG, x=333

### 底部导航 (Bottom Nav) — 仅主页面使用
- y=728, h=64
- 4 个图标: 22x22 SVG, y=740
- 图标间距: 84px (x=42, 126, 210, 294)
- 标签: 13px, y=770
- 选中态: Primary 色

### 按钮
- **主按钮**: Primary 色, 圆角 19px (pill), h=48-52, 白色文字 Semi Bold 15-16px
- **次按钮**: 描边 Primary 色, 透明背景
- **小按钮**: h=32-38, 圆角 16-19px

### 卡片
- 白色背景, 圆角 12-16px
- 可选 1px Border 色边框

### 输入框
- 白色背景, 1px Border 色边框, 圆角 12px
- h=48, 内边距 16px
- 占位符 Text-3 色

### 分割线
- 1px, Border 色

### 图标
- Lucide SVG, 20-22px
- 颜色跟随上下文 (Primary / Text-2)

## 中文文案

所有界面文字使用中文。价格使用 ¥ 符号。
